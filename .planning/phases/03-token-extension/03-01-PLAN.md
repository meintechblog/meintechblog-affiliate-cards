Create `includes/class-mtb-affiliate-token-prepass.php`:

```php
declare(strict_types=1);

final class MTB_Affiliate_Token_Prepass {
    private MTB_Affiliate_Product_Library $library;

    // Shorthand keywords (lowercase) -- NOT 10-char ASIN patterns
    private const SHORTHAND_KEYWORDS = ['last', 'heute', 'today', 'gestern', 'yesterday'];

    // Pattern matches paragraphs (same as Post Processor)
    private const PARAGRAPH_PATTERN = '/(?:<!-- wp:paragraph\b.*?-->\s*)?<p>(.*?)<\/p>(?:\s*<!-- \/wp:paragraph -->)?\s*/su';

    // Matches amazon:keyword where keyword is a shorthand (NOT a 10-char ASIN)
    private const SHORTHAND_TOKEN_PATTERN = '/^amazon:(last|heute|today|gestern|yesterday)$/i';

    // Inline shorthand pattern
    private const INLINE_SHORTHAND_PATTERN = '/\bamazon:(last|heute|today|gestern|yesterday)\b/i';
}
```

Constructor receives `MTB_Affiliate_Product_Library` via dependency injection.

Main method: `public function resolve(string $content): string`
- If content does not contain any shorthand keyword, return content unchanged (fast path).
- Use preg_replace_callback with PARAGRAPH_PATTERN to process each paragraph.
- For each paragraph, strip tags and check if the trimmed text matches SHORTHAND_TOKEN_PATTERN (standalone case).
  - If standalone match: resolve the keyword to ASINs, replace paragraph with N `<!-- wp:paragraph --><p>amazon:{ASIN}</p><!-- /wp:paragraph -->` blocks. If no products found, return empty string (removes paragraph).
  - If no standalone match: check for INLINE_SHORTHAND_PATTERN in the paragraph text.
    - If inline match found: resolve keyword to ASINs. Replace the first occurrence of `amazon:keyword` with `amazon:{FIRST_ASIN}` in the HTML. Append additional paragraphs for remaining ASINs after the current paragraph.
    - If no products found for inline token: remove the `amazon:keyword` text.
- Return modified content.

Private method: `private function resolve_keyword(string $keyword): array`
- Maps keyword to product library call:
  - `last` -> `[$this->library->get_last(1)]` (wrap single result in array, filter null)
  - `heute`, `today` -> `$this->library->get_products_today()`
  - `gestern`, `yesterday` -> `$this->library->get_products_yesterday()`
- Returns array of product rows (may be empty).
- Extracts just the ASINs: `array_map(fn($p) => $p['asin'], $products)`
