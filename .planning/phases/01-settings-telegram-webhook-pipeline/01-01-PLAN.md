**Part A: Create `includes/class-mtb-affiliate-url-resolver.php`**

Class: `final class MTB_Affiliate_Url_Resolver` with `declare(strict_types=1)`.

This class ports the ShortURL Detector + Resolver logic from flows.json (nodes `afb7be788b65b3c3` and `6a28d596b636debd`).

Methods:

1. `public function is_short_url(string $text): bool` — Returns true if text contains `https?://amzn\.(to|eu)/[^\s]+`. Port the exact regex from flows.json ShortURL Detector node.

2. `public function extract_short_url(string $text): ?string` — Extracts and returns the first `amzn.to` or `amzn.eu` URL from text using regex `/https?:\/\/amzn\.(to|eu)\/[^\s]+/i`. Returns null if no match. Port from flows.json: `raw.match(/https?:\/\/amzn\.(to|eu)\/[^\s]+/i)`.

3. `public function resolve(string $url): ?string` — Follows redirects using `wp_remote_get()` (NOT wp_safe_remote_get per PITFALLS.md Pitfall 5). Args:
   ```php
   $response = wp_remote_get($url, [
       'timeout'     => 5,
       'redirection' => 10,
       'user-agent'  => 'Mozilla/5.0 (compatible; MTB-Affiliate-Bot/1.0)',
       'sslverify'   => true,
   ]);
   ```
   If `is_wp_error($response)`, return null.
   Extract final URL from: `$response['http_response']->get_response_object()->url` if available, else check `wp_remote_retrieve_header($response, 'location')` as fallback (from flows.json ShortURL Resolver node). Return null if no final URL found.

4. `public function extract_asin(string $url): ?string` — Port the exact ASIN extraction regex from flows.json `extractAsin()`:
   ```php
   private const ASIN_PATTERN = '/\/(?:dp|gp\/product|product|exec\/obidos\/ASIN)\/(B0[A-Z0-9]{8})\b/i';
   ```
   Match against URL, return `strtoupper($match[1])` or null. NOTE: This uses B0-prefix restriction from flows.json, NOT the broader `normalize_asin()` from the REST controller.
